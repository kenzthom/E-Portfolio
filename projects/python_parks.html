Investigating Species at Risk Observations Within Canadian National Parks

Species at risk (SAR), particularly those with specialized habitat requirements, are highly vulnerable to landscape change driven by human activity and development. Protected areas are intended to mitigate these pressures by conserving critical habitats and maintaining biodiversity. However, while national parks encompass a range of ecosystems, certain habitats essential to SAR may be underrepresented within park boundaries and therefore inadequately protected. This national-scale analysis evaluates which SAR occur within Canadian national parks and assesses whether their associated habitats are sufficiently represented. For this study, the analysis focused on terrestrial mammals, with all data processing conducted in Python using Pandas, GeoPandas, and Matplotlib.

Data

  • Canadian Provincial Park Polygons

  • SAR Registry (as defined under the Species at Risk Act)

  • GBIF (Global Biodiversity Information Facility) Species Occurrence Data

# Adapted from Seely https://github.com/subornaa/Geospatial-data-analysis-project-started-kit/blob/main/parks_sar.ipynb
# Based on code from https://discourse.gbif.org/t/retrieving-inaturalist-observations-for-a-region-observers-with-pygbif/4565

species_occurrences = {}

# Adapted into a loop to download observations for each species in our target list.
for TARGET_SP in tar_species:
    # Indicate which species is being downloaded
    print(f"\nDownloading occurrences for {TARGET_SP}...")

    # Create empty list for results
    results = []
    offset = 0
    # If occurrences are for TARGET_SP, in Canada, have coordinates and no geospatial issues, assign them to out
    while True:
        out = occurrences.search(scientificName=TARGET_SP, # this seaches the GBIF database for all observations of the species in the loop.
                                 country='CA', # this narrows down the scope to Canada
                                 hasCoordinate=True,
                                 hasGeospatialIssue=False,
                                 offset=offset
                                )
        # If there are no results of occurrences, break
        if len(out['results']) == 0:
            break
        # Give an indication of progress
        results.extend(out['results'])
        offset += len(out['results'])
        print(f"Total records downloaded: {len(results)}")

    # If no reslts, continue
    if not results:
        print(f"---No data found for {TARGET_SP}")
        continue

    lat_ls = [results[i]['decimalLatitude'] for i in range(len(results))]
    lon_ls = [results[i]['decimalLongitude'] for i in range(len(results))]

    # Create dataframe of results for this species.
    df = pd.DataFrame(results)

    # Append results for this species to the occurrences_gdf
    occurrences_gdf = gpd.GeoDataFrame(
        df,
        geometry=gpd.points_from_xy(df.decimalLongitude, df.decimalLatitude),
        # Indicate df crs
        crs="EPSG:4326"
        # Transform to parks_gdf crs
    ).to_crs(parks_gdf.crs)

    # Keep only points inside parks using geopandas spatial join with intersects predicate
    in_parks_gdf = gpd.sjoin(occurrences_gdf, parks_gdf, predicate='intersects')

    # Store results of species occurrences in the parks
    species_occurrences[TARGET_SP] = in_parks_gdf

    print(f"---Points inside parks: {len(in_parks_gdf)}")

summary_with_values = summary.merge(rank_value_df, on="sara_status", how="left")

# Weighted score is calculated as the number of species multiplied by the weight of that listing level summed for the park.
summary_with_values["weighted_score"] = summary_with_values["n_species"] * summary_with_values["sara_value"]


# Create summary table with Park Scores
park_scores = (
    # Start with the DataFrame
    summary_with_values
    # Group by unique name and weighted score, sum and sort
    .groupby("NAME_E")["weighted_score"]
    .sum()
    .reset_index(name="weighted_score")
    .sort_values("weighted_score", ascending=False)
)

park_scores2 = park_scores.copy()

Objective 3: Identify species with underrepresented habitat in parks

Use point buffer as indicator of species core habitat 
# Make sure CRS of two input layers match
all_sp_occ = all_sp_occ.to_crs(parks_gdf.crs)

# Create 5000 m buffer around each species observation point
    # We selected a large buffer to capture potential habitat for each species. A large buffer reduces the impact of many occurrences on the percieved habitat size.
    # In the full analysis, we would utilize available landcover layers, and available range information to indicate what is truly "available habitat".
all_sp_occ['buffer'] = all_sp_occ.geometry.buffer(5000)

# Set buffer as the active geometry column
all_sp_occ = all_sp_occ.set_geometry('buffer')

# Dissolve buffers so area is not double counted for occurences with overlapping buffers
buffers_dissolved = all_sp_occ.dissolve(by='species', as_index = False)

# Intersect dissolved species buffers with parks polygon layer
buffers_in_parks = gpd.overlay(buffers_dissolved, parks_gdf, how='intersection')

# Calculate area in km2
buffers_in_parks['area_km2'] = buffers_in_parks.geometry.area / 1_000_000

# Create summary table
species_area_summary = (
    buffers_in_parks
    .groupby('species')['area_km2']
    .sum()
    .reset_index()
    .sort_values(by='area_km2')
)
print("The species with the least area of habitat in the National Parks are: \n", species_area_summary)
